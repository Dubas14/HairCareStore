## 2. Локалізація — тільки українська

- Прибрати 4 мови (en, pl, de, ru) з `payload.config.ts` localization
- Залишити тільки `uk` як єдину мову
- Поля `localized: true` залишаються в схемі (щоб легко повернути мультимовність)
- В UI зникнуть табки перемикання мов

### Колекції що зміняться:
Banners, Pages, PromoBlocks, BlogPosts, Categories, Brands, Products, Promotions, AutomaticDiscounts, ProductBundles, Subscribers, Ingredients

## 3. Модуль дозволів (Permissions)

### Концепція
- Дозволи налаштовуються per-user в розділі Адміністратори
- Адмін завжди має повний доступ (не обмежується)
- Для ролі "editor" — чекбокси CRUD по кожній колекції

### UI
Вкладка "Дозволи" на сторінці редагування користувача:

| Колекція          | Перегляд | Створення | Редагування | Видалення |
|-------------------|----------|-----------|-------------|-----------|
| Товари            | ✅       | ✅        | ✅          | ❌        |
| Замовлення        | ✅       | ❌        | ✅          | ❌        |
| Клієнти           | ✅       | ❌        | ❌          | ❌        |
| Банери            | ✅       | ✅        | ✅          | ✅        |
| Налаштування      | ❌       | —         | ❌          | —         |

### Реалізація
- Нове поле `permissions` (JSON) в колекції Users
- Формат: `{ "products": { "read": true, "create": true, "update": true, "delete": false }, ... }`
- Access control functions читають permissions з поточного user
- Глобали (SiteSettings, ShippingConfig тощо) — окремі read/update дозволи

## 4. Візуальна консистентність

### Єдиний layout для всіх колекцій:
- **Список**: єдиний компонент таблиці (пошук, фільтри, сортування, статус-бейджі)
- **Форма**: основні поля зліва (70%), сайдбар справа (30%) з мета-інфо
- **Кнопки**: основна (teal), другорядна (outline), небезпечна (red)
- **Статус-бейджі**: зелений (активне), жовтий (чернетка), сірий (неактивне), червоний (скасовано)

### Підказки
- Всі `description` полів — українською
- Кожне неочевидне поле має опис що це і як впливає на сайт
- LoyaltySettings — переклад описів з англійської на українську

## 5. Чистка полів

### Видалити:
- **ShippingConfig**: legacy масив `methods` (дублює `zones`)

### Перевірити та прибрати якщо не використовується:
- **SiteSettings**: аудит кожного поля — чи рендериться на фронті
- **Carts**: `stripeClientSecret`, `billingAddress`

### Перекласти на українську:
- **LoyaltySettings**: всі description полів (зараз англійською)
